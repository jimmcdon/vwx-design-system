# Design Tokens

Design tokens are the visual design atoms of the VW Xperience Design System. They are named entities that store visual design attributes such as colors, spacing, typography, and more.

## Usage

### CSS Custom Properties

```css
/* Use design tokens in your CSS */
.my-component {
  padding: var(--spacing-component-padding-md);
  font-size: var(--typography-fontSize-base);
  color: var(--theme-bw-color-text-primary);
}
```

### JavaScript/TypeScript

```typescript
import { spacing, typography } from '@vwx/tokens';

const buttonPadding = spacing.component.padding.md;
const baseFontSize = typography.fontSize.base;
```

## Theme Tokens

Theme-specific tokens are applied based on the `data-theme` attribute:

```html
<!-- B&W Theme -->
<html data-theme="bw">

<!-- Color Theme -->
<html data-theme="color">

<!-- Patina Theme -->
<html data-theme="patina">
```

Each theme provides semantic tokens for:
- Background colors (primary, secondary, tertiary)
- Text colors (primary, secondary, tertiary, inverse)
- Border colors (primary, secondary, focus)
- Interactive states (primary, hover, active)
- Status colors (success, warning, error, info)
